# ValueX7
[Git Source](https://github.com/Uniswap/twap-auction/blob/371cb0f36b92bd941e1e8d26644b52a674dda04d/src/libraries/ValueX7Lib.sol)

A ValueX7 is a uint256 value that has been multiplied by MPS

*X7 values are used for demand values to avoid intermediate division by MPS*


```solidity
type ValueX7 is uint256;
```
